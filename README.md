# AgentMira: Buyer Lead Intake Agent 🤖🏡

An autonomous, multi-agent system designed to ingest unstructured buyer inquiries, extract precise real estate constraints, and dynamically query an active MLS database to generate structured, actionable Lead Briefs for human realtors.

## 🚀 Overview

Real estate agents receive dozens of unstructured inquiries daily. This system acts as an AI teammate that reads these messages, sanitizes them against prompt injections, extracts hard and soft constraints, queries local MLS data, and hands the realtor a clean, structured summary with recommended properties and next steps.

## 🛠 Tech Stack

* **Backend:** Python, FastAPI, Pandas, LangChain, Google Gemini 2.5 Flash
* **Frontend:** React.js, Vite, Tailwind CSS
* **Architecture:** Multi-Step State Machine & Decoupled Offline Processing

## ⚡ Features

* **Security Guardrails:** Actively intercepts and flags prompt injection attacks or malicious instructions.
* **Deterministic Tooling:** Uses Pandas for programmatic CSV filtering to ensure 100% mathematical accuracy on budgets and bed counts (avoiding LLM hallucinations).
* **Dynamic Fallback Engine:** Automatically relaxes search parameters if the buyer's initial constraints return zero market matches.
* **Asynchronous Resiliency:** LLM generation is handled via a dedicated offline worker script to protect against free-tier API rate limits.

## ⚙️ Quick Start Guide

### Prerequisites

* Python 3.9+
* Node.js v18+ & npm
* A valid Google Gemini API Key

### 1. Backend Setup

Navigate to the backend directory, set up your virtual environment, and install dependencies:

```bash
cd backend
python -m venv venv

# Activate the virtual environment:
# On Mac/Linux: source venv/bin/activate
# On Windows: venv\Scripts\activate

pip install -r requirements.txt
```

Create a .env file in the backend/ folder and insert your API key:

```plaintext
GOOGLE_API_KEY="your_api_key_here"
```

Start the FastAPI server:

```bash
uvicorn main:app --reload
```

### 2. Frontend Setup

Open a new terminal window, navigate to the frontend directory, and start the Vite server:

```bash
cd frontend
npm install
npm run dev
# The React application will start at http://localhost:5173
```

### 3. Usage & Testing

The Web Dashboard: Navigate to http://localhost:5173. Click the "Process First 3 Leads" button.

Note: To ensure a zero-latency demonstration and prevent Google API rate-limit crashes during the UI review, the dashboard endpoint serves precomputed data generated by the backend AI engine.

#### The AI Engine (Offline Worker)

To evaluate the actual LangChain/Gemini integration and prompt logic, inspect and run the offline background worker in the backend directory. This script utilizes the LeadIntakeAgent to safely process raw leads with intelligent backoffs for API limits:

```bash
cd backend
python generate_briefs_offline.py
```
